# my dotFiles

## Usage

### Path:~
```zsh
git clone https://github.com/SugarShootingStar/dotFiles.git
cd dotFiles
chmod +x install.sh
sudo ./install.sh
```


## anyenv
```zsh
git clone https://github.com/riywo/anyenv ~/.anyenv
exec $SHELL -l
```

### ndenv
```zsh
any1env install ndenv
ndenv install v11.2.0
ndenv global v11.2.0
```


## exenv
```zsh
anyenv install exenv
exenv install 1.7.4
exenv global 1.7.4
```
